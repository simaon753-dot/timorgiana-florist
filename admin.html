Painel de administração

Publique fotografias e produtos diretamente no site — sem programas, a partir de qualquer aparelho.

A chave fica guardada apenas neste aparelho — nunca é enviada para outro lado além do GitHub.

Como criar a chave de acesso (uma única vez)
  1. Entre no github.com com a conta simaon753-dot.
  2. Vá a Settings → Developer settings → Personal access tokens → Fine-grained tokens e carregue em Generate new token.
  3. Token name: Painel Timorgiana · Expiration: Custom → 1 ano.
  4. Em Repository access escolha Only select repositories → timorgiana-florist.
  5. Em Permissions → Repository permissions, ponha Contents em Read and write. (Mais nada.)
  6. Generate token → copie o código github_pat_… e cole-o aqui em cima.

⚠️ Guarde a chave num sítio seguro (gestor de senhas). Se perder o aparelho, apague a chave em GitHub → Settings → Developer settings → Fine-grained tokens → Revoke. Instruções completas no ficheiro ADMIN.md.
